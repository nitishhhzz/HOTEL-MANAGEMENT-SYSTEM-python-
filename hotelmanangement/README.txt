HOTEL MANAGEMENT SYSTEM
=======================

Project Description
-------------------
This is a simple Hotel Management System developed using Flask, HTML, CSS, and Firebase Realtime Database.

Features
--------
- Book hotel rooms through a web interface
- Multiple room categories
- Real-time booking data storage in Firebase
- Displays remaining room availability
- Simple and easy-to-use interface

Room Categories
---------------
1. Single Room
2. Double Room
3. Deluxe Room
4. Suite Room

Technologies Used
-----------------
- Python
- Flask
- HTML
- CSS
- Firebase Realtime Database
- Firebase Admin SDK

Installation
------------
1. Clone the repository.
2. Create a virtual environment:
   python -m venv .venv

3. Activate the virtual environment:
   Windows:
   .venv\Scripts\activate

4. Install dependencies:
   pip install flask firebase-admin

5. Place your Firebase service account key file as:
   sec.json

6. Run the application:
   python app.py

7. Open:
   http://127.0.0.1:5000

Important
---------
Do NOT upload the following files to GitHub:
- sec.json
- .venv folder
- __pycache__ folder
